# Typecho主题[Sagiri]

### 使用该主题前，请先安装并启用点赞插件（在Plugin-Like）有

这是一款毛玻璃效果，但也有一丢丢MD风格的主题（反正规范啥的我不懂，大概就这样子就得了），可以说是非常适合萌豚建站，包括我

主题的名字和某位黄漫老师没有任何关系(笑)，请不要把这个名字与任何现实或者ACGN作品中的人物联系起来(背景和预览已经暴露了)
